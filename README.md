# Edge LaTeX 复制助手

这个扩展会在网页右上角放一个可拖动悬浮球 `TeX`：

- 先在网页中选中文字
- 点击悬浮球
- 自动复制选区内容，并尽量提取/清洗其中的 LaTeX 为 Word 更易识别的线性格式

## 已实现能力

- 悬浮球可拖动，位置会记忆
- 点击后读取当前选区
- 优先提取 MathJax/KaTeX 中常见的 `annotation` LaTeX 源码
- 自动做一层 Word 兼容化处理（如 `\dfrac -> \frac`、去掉 `\left \right`、常见数学 Unicode 转 LaTeX 命令）
- 复制成功/失败会有右上角提示

## 安装方式（开发者模式）

1. 打开 Edge，进入 `edge://extensions/`
2. 打开右上角“开发人员模式”
3. 点击“加载解压缩的扩展”
4. 选择当前目录：

```text
e:\新学习文件\学习文件\论文\针钩磨损\论文撰写\自动复制latex脚本
```

## 使用方式

1. 打开任意普通网页（`http/https` 页面）
2. 选中你要复制的文字/公式
3. 点击右上角悬浮球 `TeX`
4. 去 Word 粘贴（建议在公式输入环境中粘贴）

## 文件结构

- `manifest.json`: 扩展声明（MV3）
- `content.js`: 悬浮球、拖动、选区提取、LaTeX 转换、复制逻辑
- `content.css`: 悬浮球与提示样式

## 说明

- 浏览器安全限制下，`chrome://`、扩展商店页、部分内置页无法注入脚本，这是正常现象。
- Word 对 LaTeX 的支持是“子集”，个别复杂命令仍可能需要手工微调。
